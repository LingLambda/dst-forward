## 🌟 介绍

**Koishi Plugin DST Forward** 是一个koishi插件，能够在 **QQ** 和 **饥荒联机版**（DST）之间实现消息互通，并且允许你通过 QQ 执行回档、保存、重置等指令。  
为了实现这些功能，还需要在饥荒联机版中安装 **MOD: DST TO QQ**。
